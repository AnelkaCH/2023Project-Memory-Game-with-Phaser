# Simple Memory Game 🧠

(Note: This project was made in 2023, but I decided to upload it to GitHub in 2026 to keep it safe :D)

This project was created as a way for me to learn the Phaser game framework and gain hands-on experience with game development concepts such as player movement, keyboard input, timers, collision detection, and game state management.

## Play Online

**Live Demo:** https://the-simple-memory-game.netlify.app

## Gameplay

You have **60 seconds** to find all matching pairs hidden inside boxes around the map.

Move around, reveal items, and test your memory before time runs out.

### Controls

| Action   | Key        |
| -------- | ---------- |
| Move     | Arrow Keys |
| Open Box | Space Bar  |

To open a box, move close to it until it turns **green**, then press **Space**.

## Rules

* Only **two boxes can be opened at the same time**.
* If the revealed items match, they remain open.
* If they do not match, both boxes close again.
* One special box has **no matching pair**. When opened, it turns **red**.
* Match all available pairs before the timer reaches zero to win.

## Features

* 60-second countdown timer
* Player movement with arrow keys
* Proximity-based interaction system
* Memory matching mechanics
* Win screen
* Lose screen
* Built using Phaser

## Technologies Used

* JavaScript
* Phaser
